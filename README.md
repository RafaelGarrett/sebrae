## Requisitos

    * PHP 8.2 ou superior
    * Composer
    * Node.js 24 ou superior

## Como rodar o projeto 

    ```
    php artisan serve
    ```

    Acessar o conteúdo padrão do Laravel
    ```
    http://localhost:8000/
    ```
    Executar as migration
    ```
    php artisan migrate
    ```
    ```
    npm run dev
    ```
    ```
    docker compose up
    ```
