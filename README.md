## Requisitos

    * PHP 8.2 ou superior
    * Composer
    * Node.js 24 ou superior

## Como rodar o projeto 

    ```
    php artisan serve
    ```

    Acessar o conteúdo padrão do Laravel
    ```
    http://localhost:8000/
    ```
    ```
    docker compose up -d --build
    ```
    Executar as migration
    ```
    sudo docker exec -it sebrae-app-1 bash
    php artisan migrate
    nvm install 20
    nvm use 20
    npm run build
    npm run dev
    ```
